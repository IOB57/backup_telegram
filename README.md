# backup_telegram
Скрипт для создания резервного копирования сохраненых сообщений в telegram

Для запуска скрипта потребуется установленый python 3.9

Для работы скрипта необходимо:
Вставить Ваши значения для API_ID и API_HASH


Как получить  Api_id и Api_hash в telegram для своего аккаунта:
1. В браузере открываем страницу создания приложения для аккаунта Telegram: https://my.telegram.org/apps И вводим номер телефона аккаунта для авторизации. Нажимаем «Next».
2. В мессенджер придет код, который копируем и вставляем в соответствующее поле. Нажимаем «Next».
3. Откроется страница регистрации приложения Telegram. Заполняем её любыми понравившимися нам данными – это не имеет значения. Следуем инструкции, приведенной ниже, и жмем «Create application».
4. Откроется страница конфигурации нашего приложения. На ней нас интересуют поля Api_id и Api_hash – копируем и вставляем
